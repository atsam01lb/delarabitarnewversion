## 🚀 Deploying to GitHub Pages

1. Replace ALL files in your `atsam01lb.github.io` repo with the contents of this folder
2. Commit & push
3. GitHub Pages will rebuild automatically — wait 1–2 minutes
4. Visit:
   - `https://atsam01lb.github.io/` → Home
   - `https://atsam01lb.github.io/delar-arts/` → Delar' arts
   - `https://atsam01lb.github.io/delar-tattoo/` → Delar Tattoo
   - `https://atsam01lb.github.io/contact/` → Contact
