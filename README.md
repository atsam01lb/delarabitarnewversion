# Delara Bitar — Official Website

The official website for artist **Delara Bitar** and her two creative brands:

- 🎨 **Delar' arts** — Art studio in Amioun, North Lebanon
- 🖤 **Delar Tattoo** — Tattoo, henna & piercing studio

Built and maintained by [Colart Digital Marketing Agency](https://www.colartdigitalmarketingagency.com).

---

## 🌐 ONE Website, 4 Pages

All pages are part of the same website — just different routes:

| Page | URL Path |
|------|----------|
| Home | `/` |
| Delar' arts | `/delar-arts/` |
| Delar Tattoo | `/delar-tattoo/` |
| Contact | `/contact/` |

✅ Uses **relative paths**, so it works in any setup:
- `username.github.io/repo-name/` ✅
- `username.github.io/` ✅
- Netlify / Vercel / custom domain ✅

## 🚀 How to Deploy on GitHub Pages

1. Upload ALL files & folders to your repo
2. Go to **Settings → Pages**
3. Source: `main` branch, root folder `/`
4. Wait 1-2 minutes for deployment
5. Visit your site!
